> 在真实的渗透中，第一步便是在不惊动对方的情况下找到目标。

因为此前我能接触到的都是平台提供的靶机或客户提供的目标信息，所以在阅读渗透测试书籍时，公开信息收集部分从未被认真对待过。然而在参与 hvv 的第一天，我就陷入了找不到目标的痛苦中......终于决定好好梳理下公开信息收集的方法~

# 浏览目标网站

记录下页面中出现的：

* 人名
* 职位
* 邮箱
* 手机号
* 社交账号

## theHarvester

[Github - theHarvester](https://github.com/laramies/theHarvester)

![theharvester](/images/methodology/osint/theharvester.png)

```
theharvester -d [xxx.com] -b google
```

## 社交媒体

## Recon-ng

[Github - recon-ng](https://github.com/lanmaster53/recon-ng)

![recon-ng](/images/methodology/osint/recon-ng.png)

```
marketplace search <github>
marketplace info [module]
marketplace install [module]
modules load [module]
info
options set SOURCE xxx.com
run
back
show
show hosts
```

并没有感觉很好用，以后再深入了解吧。

# Google Hacking

[wiki - Google_hacking](https://en.wikipedia.org/wiki/Google_hacking)

[google-hacking-db](https://www.exploit-db.com/google-hacking-database)

```
site:xxx.com
filetype:php
intitle:"index of" "parent directory"
```

百度，bing 也支持这种查询方式，可以用它查找目标暴露在外的登录界面。
